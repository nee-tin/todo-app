Todo App – Interactive Task Management

Built a dynamic task management interface using vanilla JavaScript. Features include:

  • Add, edit, delete, and mark tasks as complete
  • Filter tasks by All, Active, and Completed
  • Persistent storage using localStorage
  • Responsive and user-friendly UI with subtle animations (fade-in/out empty state)
  • Clean code with state management and DOM manipulation
  • Focused on JavaScript fundamentals, user experience, and real-world UI logic

Tags: javascript, todo, vanilla-js, frontend, dom, localstorage
